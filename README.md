## Installing Instructions
- Install [ruby](https://www.ruby-lang.org/en/documentation/installation/)

In your Terminal:
```
- gem install sinatra
- gem install bundler
- Git Clone https://github.com/pmutua/ToDo-app.git
- bundle install
- cd 
- Type:< ruby app.rb >
- Run localhost:4567 in your browser

```
